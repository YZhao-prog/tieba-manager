# 贴吧吧务小工具

基于 [aiotieba](https://github.com/lumina37/aiotieba) 的贴吧吧务网页工具，**单文件、开箱即用**。

运行后自动打开浏览器，在页面里填一次凭证即可使用，无需编辑配置文件。

## 功能

| 功能 | 说明 |
| --- | --- |
| 主题帖爬取 | 输入帖子 tid，抓取全部楼层与楼中楼 |
| 用户回复查询 | 输入 tieba_uid，列出该用户全部公开回复及原帖链接 |
| 被删帖记录 | 查询某用户在指定吧吧务后台的被删帖日志（需 STOKEN） |

结果可在网页浏览，也可一键复制或下载为 `.txt`。封禁功能暂不提供，原始脚本保留在 [`legacy/`](legacy/)。

## 使用

```bash
pip3 install aiotieba      # 首次运行前装一次依赖
python3 tieba_tool.py      # 启动，浏览器会自动打开
```

浏览器打开后：

1. 右上角填入 **BDUSS**（被删帖记录还需 **STOKEN**），点「保存并登录」。
2. 选功能标签，填参数，点按钮即可。

凭证只保存在你**本机浏览器**里（localStorage），不写入任何文件、不上传。

## 凭证获取

登录网页版贴吧后，在浏览器 Cookie 中查看 `BDUSS` 与 `STOKEN`：

- **BDUSS** — 必填。
- **STOKEN** — 选填，被删帖记录查询需要，且账号须为对应吧吧务。

## 说明

- 这是本地小工具：aiotieba 需用你的凭证调用百度接口，必须在本机跑 Python，无法做成纯静态网页托管。
- 仅监听 `127.0.0.1`，只有本机能访问。
