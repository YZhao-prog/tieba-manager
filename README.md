# 贴吧吧务管理台

基于 [aiotieba](https://github.com/lumina37/aiotieba) 的贴吧吧务管理网页工具，**单文件、开箱即用**：跑起来自动打开浏览器，凭证填一次即可。后端 + 前端全部在 `tieba_tool.py` 一个文件里。

**工作流**：先在顶部输入你管理的**贴吧名并「进入」**，即可看到该吧的概览与吧务列表；处理记录就在这个吧的上下文里，不用重复填吧名。

## 功能

| 功能 | 说明 |
| --- | --- |
| 概览 | 进入某个吧后显示**吧信息**（关注/帖数/slogan）与**吧务列表**（大吧主/小吧主/各类编辑）；点某个吧务可直接查 TA 的处理记录 |
| 处理记录 | 当前吧的**删贴+封禁**记录：被查询人留空=全吧最近，填了=只看该用户；可**按操作类型/吧务/被处理人分类**、按时间排序 |
| 用户发言 | 输入主页 id，查该用户**跨吧**的**回复 / 主题帖 / 全部**（不限于当前管理的吧）；结果可**按贴吧分类**（吧名可搜，显示每吧回复/主题帖数） |

结果**边加载边显示**（流式），支持**文本搜索、翻页（含页码跳转、50/100/200 每页）、复制、下载 txt、导出网页**；同一查询有**内存缓存**，再查秒开。

> **导出网页**：生成一个自包含的 `.html` 文件，排版清晰、原帖链接可点、不含任何凭证，直接发给别人用浏览器打开即可查看。

## 使用

```bash
cd tieba-manage
python3 -m venv .venv                 # 首次：建虚拟环境
./.venv/bin/pip install aiotieba      # 首次：装依赖
./.venv/bin/python tieba_tool.py      # 启动，浏览器自动打开
```

以后每次只需最后一行。启动后访问 <http://127.0.0.1:8000>，`Ctrl+C` 停止。

## 凭证

两种方式，推荐前者：

1. **secret.py 自动登录**：`cp secret.example.py secret.py`，填入 BDUSS/STOKEN。启动后网页自动填好并登录；该文件已被 `.gitignore` 排除，不会提交。
2. **网页手填**：右上角输入后点「保存并登录」，存在浏览器 localStorage。

> 当 secret.py 有值时，它优先于浏览器里存的旧凭证。

获取方式（登录网页版贴吧后，在浏览器 Cookie 中查看）：

- **BDUSS** — 必填，所有功能都要。
- **STOKEN** — 仅「吧务处理记录」需要。**务必用 `.tieba.baidu.com` 域下的那个**，`.passport.baidu.com` 域下的会导致 302 鉴权失败。

## 常见问题

- **吧务处理记录报 302**：STOKEN 用错域（见上）、或吧名没填完整（部分吧名本身含“吧”字）、或当前账号不是该吧吧务。
- **查回复为空**：对方可能设了隐私（回复不公开），换选“主题帖”或“全部”常仍能看到内容。
- **楼中楼「查楼层」显示“未定位到”**：目标在极深楼层或超过扫描范围（楼层页上限 50、楼中楼深挖有请求预算）。

## 安全说明

服务只监听 `127.0.0.1` 且校验 Host，仅本机可访问；凭证只存在本机（secret.py / 浏览器），不上传。这是本地个人工具，请勿部署到公网。
